## Сайт поднят на Vercel, можешь посмотреть его по [ссылке](ups-2022-orapoulpe.vercel.app)





## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```
